# Little Red Book – Quotations from Chairman Mao Tse-tung

An offline Android app of the complete "Little Red Book" (Quotations from Chairman Mao Tse-tung), published by Foreign Languages Press, Peking (1966).

## Features
- Full text of all 33 chapters + Foreword
- Dark / Light mode toggle
- Search with highlighting
- Collapsible sections (expand/collapse by chapter)
- Fully offline – no internet permission required
- Gold star icon on red background
